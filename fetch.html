The API I selected returns a random would-you-rather prompt as a JSON object. I found on this website via the list of free APIs from the Reading Page on Canvas. Since the API just returns a single string (the prompt), that is the only piece of information I used. This API may be useful if you're bored - maybe on a website where things take a while to load/process, you can use this API to relieve the user of boredom as they wait. It could also be helpful in creating some sort of game website. For example, there could be a compatibility game where you and a friend answer a series of these would-you-rather questions, and the the more often you answer the same, the higher your score is. Rather than coming up with a load of questions yourself, you could just use these.
